
Lightstreamer – Load Test Toolkit
=================================

#### Specialized edition for Benchmarking the Lightstreamer Kafka Connector

# Introduction

This README will focus exclusively on the specifics of using LLTT within the scope of the [Lightstreamer Kafka Connector Benchmarking Tool](https://github.com/Lightstreamer/lightstreamer-kafka-connector-loadtest) project.
In this context, only the LLTT client component is truly required, as server-side/adapter load simulation is handled by dedicated tools in the other project.
Consequently, the client component has undergone modifications to align with the configuration requirements of the Lightstreamer Kafka Connector.
